The Aquila admin theme.

== Description ==

A clean black and gold color scheme for WordPress Admin (v4.0 onwards). Used by [Design by Mito](https://designbymito.com/) for client websites, Aquila also cleans up the admin area from unnecessary or potentially confusing items.

Many updates have been made to the admin area, including:

*		Admin area re-design.
*		"Posts" renamed to "Blog".
*   Dashboard metaboxes removed and cleaned up.
*   User "Profile" area cleaned up and simplified.
*   Aquila logos and icons.
*   Current user role added as an admin body class.
*   Removed "How are you.." from the top-right corner.
*		Re-designed login screen.
*		Items removed from admin bar.
*		New dashboard widgets.
*		WordPress and plugins support links on dashboard.
*		"Update" notifications hidden from Editors.
*		"Post Format" removed from posts.

== Installation ==

1. Upload aquila-admin-theme to the /wp-content/plugins/ directory.
2. Activate Aquila Admin Theme through the 'Plugins' menu in WordPress.
3. That's it! All settings are applied automatically and there's no need for any further configuration.

== Frequently Asked Questions ==

= Do I need to configure this plugin or change any settings? =

No. All settings are applied automatically once the plugin is activated and there are no settings that need to be configured.
